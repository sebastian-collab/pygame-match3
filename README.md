# Match 3 Game (Pygame)

A simple Match-3 puzzle game built using Python and Pygame.  
This is a basic single-file implementation using shapes instead of images.

## Features

- Match 3 or more tiles
- Swap adjacent tiles
- Automatic clearing and refill
- Gravity system
- Score counter
- Single file project

## Installation

Install pygame:

```bash
pip install pygame

python match3.py
